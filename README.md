# SP2024_ARCH20A_Housing
In response to the Palisades and Eaton fires, this 25,000-square-foot mixed-income multi-family housing project is at the 710 stub in Pasadena, California, emphasizing sustainability and disaster resiliency through using the logic of mycelium, where architecture grows like a living organism, and biodegradable materials.  

Technique: Agent-based modeling approach (Particle System) -> Simulating mycelium growth using Rhino and Grasshopper.
